# Text RPG system
We are building text-based rpg system. Who knows where it'll go?
The system is aimed to provide a dungeon crawler core functionalities such as: two-dimentional random dungeon map generation, combat system, dialogue system with NPCs, and backpack management.

## Quickstart
```
poetry install
```
To verify existing methods use this simple script:
```
./game_tester.py
```

## Requirements
  * Python 3.6 or higher
  * poetry 1.4.2
